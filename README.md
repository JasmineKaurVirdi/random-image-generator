# Random Dog Image Generator

A fun and simple web project that fetches and displays a random dog image from an external API each time a button is clicked.

## 🚀 Features

- Fetches a new dog image on every button click
- Demonstrates use of JavaScript fetch API and DOM manipulation
- Clean, responsive layout using HTML, CSS, and JavaScript

## 🌐 How It Works

- Calls https://dog.ceo/api/breeds/image/random to get a random dog image URL
- Updates the <img> element’s src with the image from the API
- Uses axios and async/await for clean asynchronous logic

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript (ES6)
- [Dog CEO API](https://dog.ceo/dog-api/)
